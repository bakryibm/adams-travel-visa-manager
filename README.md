# Adams Travel Visa Manager

نظام إدارة تأشيرات سياحية متكامل مصمم خصيصاً لشركة "أدمز ترافيل" (Adams Travel). تم بناء هذا النظام باستخدام React و TypeScript مع دعم كامل للغة العربية.

## المميزات الرئيسية

*   **لوحة تحكم شاملة (Dashboard):** عرض إحصائيات فورية عن التأشيرات النشطة، المنتهية، والإيرادات.
*   **إدارة التأشيرات:** تتبع تواريخ الإصدار، الدخول، والانتهاء بدقة.
*   **نظام التنبيهات الذكي:**
    *   تنبيهات قرب انتهاء صلاحية التأشيرة النشطة.
    *   تنبيهات قرب انتهاء مهلة الـ 90 يوماً للدخول.
*   **الذكاء الاصطناعي (Gemini AI):**
    *   استخراج بيانات العميل (الاسم، الجواز، الرقم) تلقائياً من صور/ملفات التأشيرة.
    *   إنشاء رسائل واتساب احترافية لتذكير العملاء بالتجديد أو المغادرة.
*   **إدارة التجديد:** تتبع عدد مرات التجديد (بحد أقصى 4 مرات) وحساب التواريخ تلقائياً.
*   **نظام مالي:** تتبع الإيرادات وسجل المعاملات اليومي.
*   **تخزين محلي:** يتم حفظ جميع البيانات محلياً على المتصفح (Local Storage) مع إمكانية تحميل نسخة احتياطية.

## التقنيات المستخدمة

*   React 19
*   TypeScript
*   Tailwind CSS
*   Google Gemini API (AI Features)
*   Lucide React (Icons)
*   Date-fns (Date Management)

## كيفية الاستخدام

1.  افتح التطبيق في المتصفح.
2.  انتقل إلى الإعدادات لتحديد اسم الشركة وبيانات الاتصال.
3.  ابدأ بإضافة تأشيرة جديدة عبر زر "تأشيرة جديدة" أو رفع ملف PDF.
